1) Посмотреть где я: pwd
2) Создать папку: mk dir Theory
3) Зайти в папку: cd Theory
4) Создать 3 папки: mkdir {folder1,folder2,folder3}
5) Зайти в любоую папку: cd folder1
6) Создать 5 файлов (3 txt, 2 json): touch {doc1.txt,doc2.txt,doc3.txt,doc4,json,doc5.json}
7) Создать 3 папки: mkdir {folder_1,folder_2,folder_3} 
8. Вывести список содержимого папки: ls
9) + Открыть любой txt файл: vim doc1
10) + написать туда что-нибудь, любой текст: 
i(insert) for starting work with Vim editor
write smth
11) + сохранить и выйти.
esc for enabling work with vim
:wq for saving and quit
12) Выйти из папки на уровень выше: cd ..

13) переместить любые 2 файла, которые вы создали, в любую другую папку: mv {2.txt,doc4.json} folder_2/
14) скопировать любые 2 файла, которые вы создали, в любую другую папку: cp {doc2.txt,doc5.json} folder_1
15) Найти файл по имени: find ./ doc4.json
16) просмотреть содержимое в реальном времени (команда grep) изучите как она работает: cat doc2.txt or grep option template doc2.txt
17) вывести несколько первых строк из текстового файла: head -2 doc2.txt
18) вывести несколько последних строк из текстового файла: tail -2 doc2.txt
19) просмотреть содержимое длинного файла (команда less) изучите как она работает: less doc2.txt
